## 第一题
请解释什么是面向对象，以及它解决了什么问题？

### 作答要求
- 30秒：给出核心定义 + 解决的主要问题
- 90秒：展开说明封装/继承/多态与工程价值

### 深入追问
1. 面向对象的三大特性各是什么？请用日常生活例子解释。
2. 继承和多态为什么常常要配合使用？

### 项目场景追问
请结合“支付系统”举例：不用多态会有什么问题？用了多态后怎么扩展新支付方式？

## 第二题

### 作答要求
- 30秒：讲清“值类型 vs 引用类型”
- 90秒：说明String不可变的设计收益

### 深入追问

2. `String` 的 `+` 拼接和 `StringBuilder` 的核心区别是什么？

### 项目场景追问
