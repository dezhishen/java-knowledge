# Java高级开发面试知识网络

这是一套面向零基础到中高级面试的 Java 学习教材，强调三件事：

- 先听懂：术语翻译 + 人话解释
- 再讲清：原理因果 + 方案取舍
- 最后能面：30秒短答 + 90秒完整答 + 深入追问

## 建议阅读顺序

1. 00 术语翻译与入门总览
2. 01 学习方法与面试策略
3. 02-03 打牢并发与JVM
4. 04-07 建立中间件与数据层体系
5. 08-12 进入分布式与系统设计
6. 13 执行45*6学习计划
7. 14 用问答标准答案做口述训练
8. 15 用根因视角做底层补强
9. 16 用固化提示词持续迭代教材

## 使用方式

- 目录导航：见 SUMMARY.md
- 固化生成规则：见 06-根因与固化/16-教材生成流程-固化提示词.md
- 每章建议先读“术语翻译”，再读“是什么/为什么/怎么做/高并发如何选择”。
